Convierte tu sitio WordPress en páginas estáticas para máxima velocidad y rendimiento.

== Description ==

Fast Static Cache es un plugin poderoso que genera versiones estáticas de las páginas de tu sitio WordPress, eliminando la necesidad de procesar PHP y consultas a la base de datos en cada visita. Esto resulta en tiempos de carga dramáticamente más rápidos y mejor experiencia para los usuarios.

= Características principales =

* **Generación automática de páginas estáticas** - Crea archivos HTML estáticos automáticamente
* **Compresión GZIP** - Archivos comprimidos para transferencia más rápida
* **Configuración flexible** - Excluye páginas específicas o user agents
* **Limpieza automática** - Borra caché cuando se actualiza contenido
* **Panel de administración** - Interfaz intuitiva para gestionar el caché
* **Precarga de caché** - Genera caché para páginas principales automáticamente
* **Estadísticas detalladas** - Monitorea el rendimiento del caché

= Beneficios =

* **Velocidad extrema** - Páginas hasta 10x más rápidas
* **Menor uso de recursos** - Reduce carga en servidor y base de datos
* **Mejor SEO** - Tiempos de carga rápidos mejoran el ranking
* **Experiencia de usuario superior** - Navegación más fluida
* **Escalabilidad** - Maneja más visitantes con los mismos recursos

== Installation ==

1. Sube el plugin a la carpeta `/wp-content/plugins/fast-static-cache`
2. Activa el plugin en la página de 'Plugins' de WordPress
3. Ve a 'Ajustes' > 'Fast Static Cache' para configurar las opciones
4. ¡Disfruta de un sitio más rápido!

== Frequently Asked Questions ==

= ¿Funciona con cualquier tema? =

Sí, Fast Static Cache funciona con cualquier tema de WordPress bien desarrollado.

= ¿Puedo excluir páginas específicas? =

Sí, puedes configurar páginas que no deben ser cacheadas, como carritos de compra o páginas de usuario.

= ¿Qué pasa con los usuarios logueados? =

Los usuarios logueados siempre ven la versión dinámica del sitio para mantener la funcionalidad completa.

= ¿Cómo se limpia el caché? =

El caché se limpia automáticamente cuando actualizas contenido, o puedes limpiarlo manualmente desde el panel de administración.
